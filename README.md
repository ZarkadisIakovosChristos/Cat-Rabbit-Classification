# Cat-Rabbit-Classification
This repository contains information and material for the binary task Cats and Rabbits Classification, a personal project. The purpose of the project is to use CNN in order to extract the most important pixels and be able to distinguish between cats and rabbits. The project was written in Python. The main task was choosing a good CNN architecture (evaluation with metrics like Accuracy, F1, Precision, Recall, Roc-Auc).


# The dataset can be found in: https://www.kaggle.com/datasets/muniryadi/cat-vs-rabbit
